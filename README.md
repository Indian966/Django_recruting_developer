### Hello Django
<pre>
Django 웹 서버 구축 toy project 입니다.
개발자 채용 사이트를 만들어 보았습니다.
</pre>


기술 스택 : <br>
웹 - Django <br>
서버 - AWS EC2 <br>
Front - html <br>
DB - SQLite <br>
etc - Docker & GithubAction <br>
